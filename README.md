# memoir

|名称|备注|
|---|---|
|小说回忆录|小说的人物事件等等回忆以及备忘|


# 相关资料
* spring boot
* restful API
* junit
* mongodb
* redis
* swagger2
* package -> jar 
* rabbitmq
* cors

# 发布
* 拷贝memoir-0.0.1-SNAPSHOT.jar到docker目录下
* 执行start.sh

 ```
 docker/
├── Dockerfile
└── start.sh
 ```
